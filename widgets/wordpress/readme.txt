== Description ==

Roxie is a no-code platform for building ChatGPT bots trained on custom data.

[youtube https://www.youtube.com/watch?v=Pa-cyPJzt5k]

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/roxie` directory, or install the plugin through the WordPress plugins screen directly.
2. Use the Settings->Roxie screen and click on "Link with Roxie"
